# Code of Conduct

Dieses Projekt soll freundlich, respektvoll und konstruktiv bleiben.

## Erwartetes Verhalten

- Respektvoller Umgang miteinander
- Sachliche Kritik an Code und Konzepten
- Rücksicht auf Datenschutz und sensible Bewerbungsdaten
- Hilfsbereitschaft gegenüber Menschen ohne Programmiererfahrung

## Nicht akzeptiert

- Beleidigungen, Diskriminierung oder Belästigung
- Veröffentlichung fremder personenbezogener Daten
- Kommerzielle Nutzung entgegen der Lizenz
