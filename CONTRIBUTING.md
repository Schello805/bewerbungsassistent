# Mitwirken

Danke, dass du zum Bewerbungsassistenten beitragen möchtest.

## Grundsätze

- Privacy-first: Bewerbungsdaten sind sensibel und sollen möglichst lokal bleiben.
- Nutzerkontrolle: KI-Ergebnisse sind Entwürfe, keine finalen Dokumente.
- Barrierearme UI: klare Sprache, gute Kontraste, Tastaturbedienbarkeit.
- Skalierbarkeit: Komponenten klein halten und fachliche Logik sauber trennen.

## Entwicklung

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Pull Requests

- Beschreibe kurz Problem, Lösung und Tests.
- Keine API-Keys, privaten Unterlagen oder personenbezogenen Daten committen.
- Rechtstexte nur mit klarer Kennzeichnung als Vorlage ändern.
